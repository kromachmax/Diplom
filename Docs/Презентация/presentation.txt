"Децентрализованное решение задачи о назначениях целей группе роботов".

Сущность задачи
Мы рассматриваем группу роботов, которые должны эффективно распределить между собой цели.
Каждый из роботов обладает такими характеристиками как радиус связи и область видимости.
Радиус связи робота это максимальное расстояние, на котором робот может:
• Обмениваться данными с другими роботами
• Получать/передавать информацию о целях
Область видимости робота это пространство, где робот может:
• Обнаруживать цели (сенсорами)
• Определять их координаты с определенными погрешностями
Погрешность определения координат целей — это разница между реальным положением цели
 и измеренными координатами, которые получает робот с помощью своих сенсоров.


Прежде чем перейти к конкретным методам важно понять архитектурные подходы к распределению 
задач между роботами. Все существующие робототехнические системы можно классифицировать по трем ключевым признакам:
1) По типу управления роботами
2) По типу связи
3) По типу видимости
